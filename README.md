# ⚡ NESCO Prepaid Meter Card Generator

A professional web-based tool to generate NESCO prepaid meter cards from Excel/JSON data with QR code, auto layout, and print-ready design.

---

## 🚀 Features

- 📂 Upload Excel / JSON file
- 🔄 Auto column mapping
- 🔍 Smart exact search (SL, Mobile, Consumer, Meter)
- 🧾 Card generation with clean UI
- 🏠 House / 🏪 Shop detection (auto)
- 🔳 QR Code generation
- 🖨️ Print optimized layout (A4)
- 🎯 Auto font resize (dynamic text fit)

---

## 📁 Supported Fields

| Field | Description |
|------|------------|
| SL | Serial Number |
| Consumer Num | Consumer ID |
| User Name | Customer Name |
| Meter Num | Meter Number |
| Mobile | Phone Number |
| Address | Location |
| Refff | Reference |
| Re-mark | Remarks |
| Type | বাড়ি / দোকান |
| House name | House Name |
| Shop name | Shop Name |

---

## 📦 How to Use

1. Upload Excel or JSON file  
2. Map columns correctly  
3. Click **Generate Cards**  
4. Search (optional)  
5. Click **Print**

---

## 🖨️ Print Settings

- Paper: A4  
- Margin: None / Minimum  
- Scale: 100%  
- Background Graphics: ON
